== Move to {ilm} step API
++++
<titleabbrev>Move to step</titleabbrev>
++++

Move {ilm} into the specified action step.

[float]
=== Request

`POST /_ilm/move/<step-name>`

[float]
=== Description

Enables you to manually move {ilm} between steps while it's performing
an action.

== Get {ilm} policy API
++++
<titleabbrev>Get policy</titleabbrev>
++++

Retrieve one or more {ilm} policies.

[float]
=== Request

`GET /_ilm/<policy_name>` +

`GET /_ilm/`

[float]
=== Description

Returns the following information for each requested policy:

* The policy phases, triggers, and actions.
* Version
* Modified date

== Set {ilm} policy API
++++
<titleabbrev>Set policy</titleabbrev>
++++

Apply an {ilm} policy to an existing index.

[float]
=== Request

`PUT /<index-name>/_ilm/<policy_name>`

[float]
=== Description

Configures an index to use the specified policy.

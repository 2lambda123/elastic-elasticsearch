=== Get information for API key(s) owned by authenticated user API

API Key(s) information owned by authenticated user can be retrieved using this API.

[id="{upid}-{api}-request"]
==== Get My API Key Request
The +{request}+ supports retrieving API key information for

. A specific API key by api key id or by name

. All API keys of current logged in user

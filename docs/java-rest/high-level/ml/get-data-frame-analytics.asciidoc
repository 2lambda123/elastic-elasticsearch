=== Get Data Frame Analytics API

The Get Data Frame Analytics API is used to get one or more {dataframe-analytics}s.
The API accepts a +{request}+ object and returns a +{response}+.

[id="{upid}-{api}-request"]
==== Get Data Frame Analytics Request

A +{request}+ requires either a {dataframe-analytics} id, a comma separated list of ids or
the special wildcard `_all` to get all {dataframe-analytics}s.

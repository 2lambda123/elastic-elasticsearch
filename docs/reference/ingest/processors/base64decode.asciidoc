[[base64decode-processor]]
=== Base64 Decode processor
++++
<titleabbrev>Base64 Decode</titleabbrev>
++++

Decodes a base64 encoded field in the currently ingested document.
If the field value is an array, all members will be decoded.

[[base64decode-options]]
.Base64Decode Options
[options="header"]
|======
| Name             | Required  | Default  | Description
| `field`          | yes       | -        | The field whose value is to be decoded
| `target_field`   | no        | `field`  | The field to assign the decoded value to, by default `field` is updated in-place
| `skip_failure`   | no        | `false`  | If `true`, the processor skips failed conversions from the input array.
| `ignore_missing` | no        | `false`  | If `true` and `field` does not exist or is `null`, the processor quietly exits without modifying the document
include::common-options.asciidoc[]
|======

[source,js]
--------------------------------------------------
PUT _ingest/pipeline/my-pipeline-id
{
  "description": "decodes the base64 string of the `input.field` and stores it in `output.field`",
  "processors" : [
    {
      "base64decode" : {
        "field" : "input.field",
        "target_field": "output.field",
        "skip_failure": true
      }
    }
  ]
}
--------------------------------------------------

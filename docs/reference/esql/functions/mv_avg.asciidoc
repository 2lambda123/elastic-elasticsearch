[[esql-mv_avg]]
=== `MV_AVG`

*Syntax*

[source,esql]
----
MV_AVG(column)
----

*Parameters*

`column`::
Column to convert from multiple values to single value.

*Description*

Converts a multivalued column into a single valued column containing the average
of all of the values.

*Supported types*

The output type is always a `double` and the input type can be any number.

[[esql-to_geopoint]]
=== `TO_GEOPOINT`
Converts an input value to a `geo_point` value.

The input can be a single- or multi-valued field or an expression.
The input type must be a string or long integer.

A string will only be successfully converted if it's respecting the WKT format:

----

include::{esql-specs}/spatial.csv-spec[tag=to_geopoint-str-result]
|===

the Lucene encoded value, with latitude encoded in the high four bytes,
and longitude in the low four bytes:

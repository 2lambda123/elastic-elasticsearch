=== `MV_DEDUPE`

*Syntax*

[.text-center]
image::esql/functions/signature/mv_dedupe.svg[Embedded,opts=inline]

*Parameters*

`v`::
Column to deduplicate.

*Description*

Removes duplicates from a multivalued column.

NOTE: `MV_DEDUPE` may, but won't always, sort the values in the column.

[[esql-mv_median]]
=== `MV_MEDIAN`

[source,esql]
----
MV_MEDIAN(column)
----

*Parameters*

`column`::
Column to convert from multiple values to single value.

*Description*

Converts a multivalued column into a single valued column containing the median
value.

*Supported types*

It can be used by any numeric column type and returns a value of the same type.

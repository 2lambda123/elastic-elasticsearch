[[esql-agg-count]]
=== `COUNT`

*Syntax*

[source,esql]
----
COUNT([expression])
----

*Parameters*

`expression`::
Expression that outputs values to be counted. If omitted, returns a count all
(the number of rows).

*Description*

Returns the total number (count) of input values.

*Supported types*

Can take any field type as input.

[[esql-agg-avg]]
=== `AVG`

*Syntax*

[source,esql]
----
AVG(expression)
----

`expression`::
Numeric expression. If `null`, the function returns `null`.

*Description*

The average of a numeric expression.

*Supported types*

The result is always a `double` no matter the input type.

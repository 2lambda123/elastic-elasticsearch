=== `MV_MAX`

*Syntax*

[.text-center]
image::esql/functions/signature/mv_max.svg[Embedded,opts=inline]

*Parameters*

`v`::
Column to convert from multiple values to single value.

*Description*

Converts a multivalued column into a single valued column containing the maximum
value.

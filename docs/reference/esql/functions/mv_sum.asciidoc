[[esql-mv_sum]]
=== `MV_SUM`

[source,esql]
----
MV_SUM(column)
----

*Parameters*

`column`::
Column to convert from multiple values to single value.

*Description*

Converts a multivalued column into a single valued column containing the sum
of all of the values.

*Supported types*

The input type can be any number and the output type is the same as the input
type.

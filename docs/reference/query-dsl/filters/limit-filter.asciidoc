[[query-dsl-limit-filter]]
=== Limit Filter

ifdef:asciidoctor[]
deprecated[1.6.0, "Use <<search-request-body,terminate_after>> instead"]
endif:[]
ifndef:asciidoctor[]
deprecated[1.6.0, Use <<search-request-body,terminate_after>> instead]
endif:[]

A limit filter limits the number of documents (per shard) to execute on.
For example:

[source,js]
--------------------------------------------------
{
    "filtered" : {
        "filter" : {
             "limit" : {"value" : 100}
         },
         "query" : {
            "term" : { "name.first" : "shay" }
        }
    }
}
--------------------------------------------------

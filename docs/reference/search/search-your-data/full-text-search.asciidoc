[[full-text-search]]
== Full-text search

Full-text search enables you to search within full-text fields in order to find
the <<full-text-relevance,most relevant>> documents. Contrary to
<<term-level-queries,term-level queries>> that typically look for a precise
values in structured data, full text queries enable you to search
<<full-text-text-analysis,analyzed text fields>>.

[[full-text-text-analysis]]
=== Text analysis

Text analysis is the processing that happens on <<text,`text` fields>>. The text
in `text` fields is broken down into smaller chunks called tokens. In most
cases, these tokens are individual words.

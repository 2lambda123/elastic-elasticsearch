[[search-request-query]]
=== Query

The query element within the search request body allows to define a
query using the <<query-dsl,Query DSL>>.

[source,js]
--------------------------------------------------
GET /_search?include_type_name=false
{
    "query" : {
        "term" : { "user" : "kimchy" }
    }
}
--------------------------------------------------
// CONSOLE

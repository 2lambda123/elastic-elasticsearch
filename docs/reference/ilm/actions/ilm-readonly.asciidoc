[[ilm-readonly]]
=== Read-Only

Phases allowed: warm.

Sets an index as read-only. See <<dynamic-index-settings,`index.blocks.write`>>.

This action does not have any options associated with it.

[source,console]
--------------------------------------------------
PUT _ilm/policy/my_policy
{
  "policy": {
    "phases": {
      "warm": {
        "actions": {
          "readonly" : { }
        }
      }
    }
  }
}
--------------------------------------------------

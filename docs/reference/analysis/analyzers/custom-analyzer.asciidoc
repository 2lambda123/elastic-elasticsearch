=== Custom Analyzer

A `custom` analyzer wraps a tokenizer and zero or more character and token 
filters. Creating a custom tokenizer enables you to select the tokenizer and
filters you want to use to analyze text. If you specify multiple character or token 
filters, the filters are applied in the order in which you list them. 

You can configure the following settings for a `custom` analyzer:

[cols="<,<",options="header",]
|=======================================================================
|Setting |Description
|`char_filter` |The character filter(s) to use, specified as an array of filter
names. Optional.
|`filter` |The token filter(s) to use, specified as an array of filter names.
Optional.
|`tokenizer` |The tokenizer to use. This can be one of the built-in
tokenizers, or a tokenizer you have configured. Required.
|=======================================================================

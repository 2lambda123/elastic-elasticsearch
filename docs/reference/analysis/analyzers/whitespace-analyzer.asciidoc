[[analysis-whitespace-analyzer]]
=== Whitespace Analyzer

The `whitespace` analyzer wraps the <<analysis-whitespace-tokenizer,
`whitespace`>> tokenizer, which splits tokens only on whitespace boundaries 
such as spaces and tabs. 

The `whitespace` analyzer has no configurable attributes.

For example, using the `whitespace` analyzer to process the string 
"Let the wild rumpus start! (bit.ly/1wsCiKv)" generates the following tokens:

 `Let, the, wild, rumpus, start!, (bit.ly/1wsCiKv)`

=== Simple Analyzer

The `simple` analyzer wraps the 
<<analysis-lowercase-tokenizer,`lowercase`>> tokenizer, which splits text
at non-letters and converts them to lowercase.

The `simple` analyzer has no configurable attributes.

For example, using the simple analyzer to process the string "Let the wild 
rumpus start! (bit.ly/1wsCiKv)" generates the following tokens:

`let, the, wild, rumpus, start, bit.ly, 1wscikv`
